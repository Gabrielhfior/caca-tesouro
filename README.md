# Caça ao Tesouro 🎁

Projeto em Python desenvolvido para praticar lógica de programação através de um jogo simples de adivinhação.

## 📌 Sobre o projeto

Neste jogo, existem cinco caixas misteriosas. Uma delas esconde um tesouro e o objetivo do jogador é encontrá-lo.

As caixas abertas incorretamente são marcadas com um emoji de explosão, enquanto a caixa correta revela uma estrela.

## ⚙️ Funcionalidades

- Sorteio aleatório da caixa vencedora
- Sistema de sequência de vitórias (streak)
- Contador de partidas jogadas
- Feedback visual utilizando emojis
- Menu simples no terminal
- Reinício automático após cada rodada

## 🧠 Conceitos praticados

- Variáveis
- Estruturas condicionais (`if`, `elif`, `else`)
- Laços de repetição (`while`)
- Listas
- Alteração de elementos em listas
- Entrada de dados (`input`)
- Geração de números aleatórios (`random`)
